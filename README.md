Youtube API
===========

A Node.JS module, which provides an object oriented wrapper for the Youtube v3 API.


## Reference
 - Activities
 - ChannelBanners
 - Channels
 - GuideCategories
 - PlaylistItems
 - Playlists
 - Search
 - Subscriptions
 - Thumbnails
 - VideoCategories
 - Videos
